## Usage
1. 更改```src/config```目录下配置文件，配置管理后端接口地址
2. 安装依赖
    > ```bash
    > npm install
    > ```

3. 以开发模式编译运行
    > ```bash
    > npm run dev
    > ```

4. 编译并压缩产生正式代码
    > ```bash
    > npm run build
    > ```

5. 代码格式化
    > ```bash
    > npm run lint
    > ```
